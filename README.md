### 栈区
**编译器分配**

不能返回局部变量的指针的原因是：

局部变量在函数结束时被销毁，虽然可以取到他的地址，但是地址对应的数值却不是原来的

地址虽在，但是地址对应住的人换了

一个函数结束时才会销毁局部变量，main函数一般不会出现返回局部变量地址发生错误的问题，因为main函数是某个项目的唯一主函数

**解决方法是：1.static修饰  or  堆区开辟  or  ->跨越阶级才能保证生存！**
